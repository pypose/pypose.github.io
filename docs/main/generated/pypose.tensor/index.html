pypose.tensor¶

pypose.tensor(inputs)[source]¶

Convert a LieTensor into a torch.Tensor without changing data.

Parameters:

inputs (LieTensor) – the input LieTensor.

Returns:

the torch.Tensor form of LieTensor.

Return type:

Tensor

Example

>>> x = pp.randn_SO3(2)
>>> x.tensor()
tensor([[ 0.1196,  0.2339, -0.6824,  0.6822],
        [ 0.9198, -0.2704, -0.2395,  0.1532]])
