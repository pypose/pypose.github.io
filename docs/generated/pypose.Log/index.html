pypose.Log

pypose.Log(input)[source]

The Logarithm map for LieTensor (Lie Group).

\[\mathrm{Log}: \mathcal{G} \mapsto \mathcal{g} \]

Note

This function Log() is different from log(), which returns a new torch tensor with the logarithm of the elements of the input tensor.

  • If input \(\mathbf{x}\)’s ltype is SO3_type (input \(\mathbf{x}\) is an instance of SO3()):

    Let \(w\), \(\boldsymbol{\nu}\) be the scalar and vector parts of \(\mathbf{x}\), respectively; \(\mathbf{y}\) be the output.

    If \(\|\boldsymbol{\nu}\| > \text{eps}\):

    \[\mathbf{y} = \left\{ \begin{array}{ll} 2\frac{\mathrm{atan}(\|\boldsymbol{\nu}\|/w)}{\| \mathbf{\nu}\|}\boldsymbol{\nu}, \quad \|w\| > \text{eps} \\ \pm\frac{\pi}{\|\boldsymbol{\nu}\|}\boldsymbol{\nu}, \quad \|w\| \leq \text{eps} \end{array} \right. \]

    otherwise:

    \[\mathbf{y} = 2\left( \frac{1}{w} - \frac{\|\boldsymbol{\nu}\|^2}{3w^3}\right)\boldsymbol{\nu}. \]

  • If input \(\mathbf{x}\)’s ltype is SE3_type (input \(\mathbf{x}\) is an instance of SE3()):

    Let \(\mathbf{t}\), \(\mathbf{q}\) be the translation and rotation parts of \(\mathbf{x}\), respectively; \(\mathbf{y}\) be the output.

    \[\mathbf{y} = \left[\mathbf{J}^{-1}\mathbf{t}, \mathrm{Log}(\mathbf{q}) \right], \]

    where \(\mathrm{Log}\) is the Logarithm map for SO3_type input and \(\mathbf{J}\) is the Jacobian matrix.

  • If input \(\mathbf{x}\)’s ltype is RxSO3_type (input \(\mathbf{x}\) is an instance of RxSO3()):

    Let \(\mathbf{q}\), \(s\) be the rotation and scale parts of \(\mathbf{x}\), respectively; \(\mathbf{y}\) be the output.

    \[\mathbf{y} = \left[\mathrm{Log}(\mathbf{q}), \log(s) \right]. \]

  • If input \(\mathbf{x}\)’s ltype is Sim3_type (input \(\mathbf{x}\) is an instance of Sim3()):

    Let \(\mathbf{t}\), \(\mathbf{q}_s\) be the translation and RxSO3 parts of \(\mathbf{x}\), respectively; \(\mathbf{y}\) be the output.

    \[\mathbf{y} = \left[\mathbf{J}_s^{-1}\mathbf{t}, \mathrm{Log}(\mathbf{q}_s) \right], \]

    where \(\mathbf{J}_s\) is the similarity transformed Jacobian matrix.

Note

The \(\mathrm{atan}\)-based Logarithm map implementation thanks to the paper:

  • C. Hertzberg et al., Integrating Generic Sensor Fusion Algorithms with Sound State Representation through Encapsulation of Manifolds, Information Fusion, 2013.

Intuitively, \(\boldsymbol{\nu}\) is used to define the rotation axis and \(w\) to define the angle of rotation \(\theta\), then the quaternion with unit norm can be written as

\[\mathbf{q} = \left[\sin(\theta/2) \mathbf{n}, \cos(\theta/2) \right] \]

where \(\mathbf{n}\) is the unit vector along the rotation axis.

Therefore, \(\|\boldsymbol{\nu}\| = \sin(\theta/2)\), \(\mathrm{atan}(\|\boldsymbol{\nu}\|/w) = \theta/2\)

Example

  • \(\mathrm{Log}\): SO3 \(\mapsto\) so3

>>> x = pp.randn_SO3(2)
>>> x.Log() # equivalent to: pp.Log(x)
so3Type LieTensor:
tensor([[-0.3060,  0.2344,  1.2724],
        [ 0.3012, -0.6817,  0.1187]])

  • \(\mathrm{Log}\): SE3 \(\mapsto\) se3

>>> x = pp.randn_SE3(2)
>>> x.Log() # equivalent to: pp.Log(x)
se3Type LieTensor:
tensor([[ 0.2958, -0.0840, -1.4733,  0.7004,  0.4483, -0.9009],
        [ 0.0850, -0.1020, -1.2616, -1.0524, -1.2031,  0.8377]])

  • \(\mathrm{Log}\): RxSO3 \(\mapsto\) rxso3

>>> x = pp.randn_RxSO3(2)
>>> x.Log() # equivalent to: pp.Log(x)
rxso3Type LieTensor:
tensor([[-1.3755,  0.3525, -2.2367,  0.5409],
        [ 0.5929, -0.3250, -0.7394,  1.0965]])

  • \(\mathrm{Log}\): Sim3 \(\mapsto\) sim3

>>> x = pp.randn_Sim3(2)
>>> x.Log() # equivalent to: pp.Log(x)
sim3Type LieTensor:
tensor([[-0.1747, -0.3698,  0.2000,  0.1735,  0.6220,  1.1852, -0.6402],
        [-0.8685, -0.1717,  1.2139, -0.8385, -2.2957, -1.9545,  0.8474]])
