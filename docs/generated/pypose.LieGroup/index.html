Examples

>>> import torch
>>> import pypose as pp
>>> data = torch.randn(3, 3, requires_grad=True, device='cuda:0')
>>> pp.LieGroup(data, gtype=pp.so3_type)
so3Type Group:
tensor([[ 0.9520,  0.4517,  0.5834],
        [-0.8106,  0.8197,  0.7077],
        [-0.5743,  0.8182, -1.2104]], device='cuda:0', grad_fn=<AliasBackward0>)

>>> pp.so3(data)  ## Alias is Recommended
so3Type Group:
tensor([[ 0.9520,  0.4517,  0.5834],
        [-0.8106,  0.8197,  0.7077],
        [-0.5743,  0.8182, -1.2104]], device='cuda:0', grad_fn=<AliasBackward0>)

Note

Alias of LieGroup for specific gtype is recommended. For example, the following usage is equivalent:

  • pp.LieGroup(data, gtype=pp.so3_type)

  • pp.so3(data)

Note

All properties of torch.tensor is available. See more details at https://pytorch.org/docs/stable/tensor_attributes.html
