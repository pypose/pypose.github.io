pypose.Jinvp

pypose.Jinvp(input, p)[source]

The dot product between left Jacobian inverse at the point given by input (Lie Group) and second point (Lie Algebra).

\[\mathrm{Jinvp}: (\mathcal{G}, \mathcal{g}) \mapsto \mathcal{g} \]
Parameters

  • input (LieTensor) – the input LieTensor (Lie Group)

  • p (LieTensor) – the second LieTensor (Lie Algebra)

Returns

the output LieTensor (Lie Algebra)

Return type

LieTensor

rxso3_type

Let the input be (\(\mathbf{x}\), \(\mathbf{a}\)), \(\mathbf{y}\) be the output.

\[\mathbf{y}_i = \mathbf{J}^{-1}_i(\mathbf{x}_i)\mathbf{a}_i, \]

where \(\mathbf{J}^{-1}_i(\mathbf{x}_i)\) is the left-Jacobian of \(\mathbf{x}_i\).

  • If input (\(\mathbf{x}\), \(\mathbf{a}\))’s ltype are SO3_type and so3_type (input \(\mathbf{x}\) is an instance of SO3(), \(\mathbf{a}\) is an instance of so3()). Let \(\boldsymbol{\phi}_i = \theta_i\mathbf{n}_i\) be the corresponding Lie Algebra of \(\mathbf{x}_i\), \(\boldsymbol{\Phi}_i\) be the skew matrix of \(\boldsymbol{\phi}_i\):

    \[\mathbf{J}^{-1}_i(\mathbf{x}_i) = \mathbf{I} - \frac{1}{2}\boldsymbol{\Phi}_i + \mathrm{coef}\boldsymbol{\Phi}_i^2 \]

    where \(\mathbf{I}\) is the identity matrix with the same dimension as \(\boldsymbol{\Phi}_i\), and

    \[\mathrm{coef} = \left\{ \begin{array}{ll} \frac{1}{\theta^2} - \frac{\cos{\frac{\theta}{2}}}{2\theta\sin{\frac{\theta}{2}}}, \quad \|\theta\| > \text{eps}, \\ \frac{1}{12}, \quad \|\theta\| \leq \text{eps} \end{array} \right. \]

  • If input (\(\mathbf{x}\), \(\mathbf{a}\))’s ltype are SE3_type and se3_type (input \(\mathbf{x}\) is an instance of SE3(), \(\mathbf{a}\) is an instance of se3()). Let \(\boldsymbol{\phi}_i = \theta_i\mathbf{n}_i\) be the corresponding Lie Algebra of the SO3 part of \(\mathbf{x}_i\), \(\boldsymbol{\tau}\) be the Lie Algebra of the translation part of \(\mathbf{x}_i\); \(\boldsymbol{\Phi}\) and \(\boldsymbol{\Tau}\) be the skew matrices, respectively:

    \[\mathbf{J}^{-1}_i(\mathbf{x}_i) = \left[ \begin{array}{cc} \mathbf{J}_i^{-1}(\boldsymbol{\Phi}) & -\mathbf{J}_i^{-1} (\boldsymbol{\Phi})\mathbf{Q}(\boldsymbol{\tau}, \boldsymbol{\phi})\mathbf{J}_i^{-1}(\boldsymbol{\Phi}) \\ \mathbf{0} & \mathbf{J}_i^{-1}(\boldsymbol{\Phi}) \end{array} \right] \]

    where \(\mathbf{J}_i^{-1}(\boldsymbol{\Phi})\) is the left Jacobian of the SO3 part of \(\mathbf{x}_i\). \(\mathbf{Q}_i(\boldsymbol{\tau}, \boldsymbol{\phi})\) is

    \[\begin{align} \mathbf{Q}_i(\boldsymbol{\tau}, \boldsymbol{\phi}) = \frac{1}{2}\boldsymbol{\Tau} &+ c_1 (\boldsymbol{\Phi\Tau} + \boldsymbol{\Tau\Phi} + \boldsymbol{\Phi\Tau\Phi}) \\ &+ c_2 (\boldsymbol{\Phi^2\Tau} + \boldsymbol{\Tau\Phi^2} - 3\boldsymbol{\Phi\Tau\Phi})\\ &+ c_3 (\boldsymbol{\Phi\Tau\Phi^2} + \boldsymbol{\Phi^2\Tau\Phi}) \end{align}\]

    where,

    \[c_1 = \left\{ \begin{array}{ll} \frac{\theta - \sin\theta}{\theta^3}, \quad \|\theta\| > \text{eps}, \\ \frac{1}{6}-\frac{1}{120}\theta^2, \quad \|\theta\| \leq \text{eps} \end{array} \right. \]
    \[c_2 = \left\{ \begin{array}{ll} \frac{\theta^2 +2\cos\theta - 2}{2\theta^4}, \quad \|\theta\| > \text{eps}, \\ \frac{1}{24}-\frac{1}{720}\theta^2, \quad \|\theta\| \leq \text{eps} \end{array} \right. \]
    \[c_3 = \left\{ \begin{array}{ll} \frac{2\theta - 3\sin\theta + \theta\cos\theta}{2\theta^5}, \quad \|\theta\| > \text{eps}, \\ \frac{1}{120}-\frac{1}{2520}\theta^2, \quad \|\theta\| \leq \text{eps} \end{array} \right. \]

Note

\(\mathrm{Jinvp}\) is usually used in the Baker-Campbell-Hausdorff formula (BCH formula) when performing LieTensor multiplication. One can refer to this paper for more details:

  • J. Sola et al., A micro Lie theory for state estimation in robotics, arXiv preprint arXiv:1812.01537 (2018).

In particular, Eq.(146) is the math used in the SO3_type, so3_type scenario; Eq.(179b) and Eq.(180) are the math used in the SE3_type, se3_type scenario.

Example

  • \(\mathrm{Jinvp}\): (SO3, so3) \(\mapsto\) so3

>>> x = pp.randn_SO3(2)
>>> p = pp.randn_so3(2)
>>> x.Jinvp(p) # equivalent to: pp.Jinvp(x, a)
so3Type LieTensor:
tensor([[ 0.8782,  0.5898, -1.9071],
        [-0.6499, -0.3977,  0.8115]])

  • \(\mathrm{Jinvp}\): (SE3, se3) \(\mapsto\) se3

>>> x = pp.randn_SE3(2)
>>> p = pp.randn_se3(2)
>>> x.Jinvp(p) # equivalent to: pp.Jinv(x, a)
se3Type LieTensor:
tensor([[-1.3803,  0.7891, -0.4268,  0.6917, -0.2167,  0.3333],
        [-1.4517, -0.8059,  0.9343,  1.7398,  0.6579,  0.4785]])
