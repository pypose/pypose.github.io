pypose.LieTensor

class pypose.LieTensor(*data, ltype)[source]

The Base Class for LieTensor

Returns

LieTensor in form of Lie Type (Inherited from torch.Tensor)

Parameters

  • data (Tensor, or list, or ‘int…’) –

    A Tensor object, or constructing a Tensor object from list, which defines tensor data, or from ‘int…’, which defines tensor shape.

    The shape of Tensor object should be compatible with Lie Type ltype, otherwise error will be raised. See following table for more details.

  • ltype (ltype) – Lie Type, either Lie Group or Lie Algebra is listed below:

Note

Two attributs shape and lshape are available for LieTensor. The only differece is that the lshape ignore the last dimension of shape.

Examples

>>> import torch
>>> import pypose as pp
>>> data = torch.randn(3, 3, requires_grad=True, device='cuda:0')
>>> pp.LieTensor(data, ltype=pp.so3_type)
so3Type LieTensor:
tensor([[ 0.9520,  0.4517,  0.5834],
        [-0.8106,  0.8197,  0.7077],
        [-0.5743,  0.8182, -1.2104]], device='cuda:0', grad_fn=<AliasBackward0>)

Alias for specific LieTensor is recommended:

>>> pp.so3(data)
so3Type LieTensor:
tensor([[ 0.9520,  0.4517,  0.5834],
        [-0.8106,  0.8197,  0.7077],
        [-0.5743,  0.8182, -1.2104]], device='cuda:0', grad_fn=<AliasBackward0>)

Other available constructors:

>>> pp.so3([0, 0, 0])
so3Type LieTensor:
tensor([0., 0., 0.])

>>> pp.so3(2, 3)
so3Type LieTensor:
tensor([[0., 0., 0.],
        [0., 0., 0.]])

Note

Alias class for LieTensor is recommended. For example, the following usage is equivalent:

  • pp.LieTensor(tensor, ltype=pp.so3_type)

  • pp.so3(tensor) (This is preferred).

Note

LieTensor constructor from tensor is recommended, since many useful attributes from torch.tensor, such as dtype, device, and requires_grad as can be used. See more details at tensor attributes.

>>> data = torch.randn(1, 3, dtype=torch.float64, device="cuda", requires_grad=True)
>>> pp.so3(data)
so3Type LieTensor:
tensor([[-1.5948,  0.3113, -0.9807]], device='cuda:0', dtype=torch.float64,
    grad_fn=<AliasBackward0>)
