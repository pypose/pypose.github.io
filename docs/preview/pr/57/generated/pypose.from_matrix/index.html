pypose.from_matrix

pypose.from_matrix(mat, ltype)[source]

Convert batched 3x3 or 3x4 or 4x4 matrices to LieTensor.

Parameters

  • mat (Tensor) – the matrix to convert.

  • ltype (class) – one of pypose.SO3(), pypose.SE3(), pypose.Sim3() or pypose.RxSO3().

Returns

the converted LieTensor.

Return type

LieTensor
