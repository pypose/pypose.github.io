pypose.mat2RxSO3

pypose.mat2RxSO3(mat)[source]

Convert batched 3x3 or 3x4 or 4x4 matrices to RxSO3Type LieTensor.

Parameters

mat (Tensor) – the matrix to convert.

Returns

the converted RxSO3Type LieTensor.

Return type

LieTensor

Shape:

Input: (*, 3, 3) or (*, 3, 4) or (*, 4, 4)

Output: (*, 5)

Suppose the input transformation matrix \(\mathbf{R}_i\) is

\[\mathbf{R}_i = \begin{bmatrix} sR_{11} & sR_{12} & sR_{13}\\ sR_{21} & sR_{22} & sR_{23}\\ sR_{31} & sR_{32} & sR_{33} \end{bmatrix}, \]

Output LieTensor should be of format:

\[\mathrm{vec}[*, :] = [q_x, q_y, q_z, q_w, s] \]

where \(\begin{pmatrix} q_x & q_y & q_z & q_w \end{pmatrix}^T\) is computed by pypose.mat2SO3().

Examples

>>> input = torch.tensor([[0., -0.5, 0.],
...                       [0.5, 0., 0.],
...                       [0., 0., 0.5]])
>>> pp.mat2RxSO3(input)
RxSO3Type LieTensor:
tensor([0.0000, 0.0000, 0.7071, 0.7071, 0.5000])

See pypose.RxSO3() for more details of the output LieTensor format.
