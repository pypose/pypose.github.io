pypose.module.EKF

class pypose.module.EKF(model)[source]

Linearized Extended Kalman Filter.

Parameters

model (System) – The noisy system model to estimate using this filter.

forward(x, y, P, U)[source]

Calculates filter one timestep.

Parameters

  • x (Tensor) – estimated system state of previous timestep

  • y (Tensor) – system observation at current timestep (measurement)

  • P (Tensor) – state covariance of previous timestep

  • U (Tensor) – system input at current timestep

Returns

estimated system state at current timestep (posteriori state estimate) P (Tensor): estimated state covariance at current timestep (posteriori covariance)

Return type

x (Tensor)

Uses a Linearized System Model, thus it is a Linearized Extended Kalman Filter. If not defined, System.A, System.B, System.C are computed each call using torch.autograd.functional.jacobian(), see System

Note

To use this filter, your System subclass must define Q (process noise covariance) and R (measurement noise covariance). Your System should add noise to the returned state and observation by the following equations:

\[w_k \sim (0, Q) \\ x_k = x_k + w_k \]
\[v_k \sim (0, R) \\ y_k = y_k + v_k \]

For each forward() call, the Kalman Filter Equations are executed as follows:

Note

Since System.x is a row-vector, before doing these calculations, x is transposed into a column vector.

  1. Predict State (Priori State Estimate) using State Transition Equation.

\[\mathbf{x} = f(x_{k-1}, U_k) \]

For a Linear System, that is:

\[\mathbf{x} = \mathbf{A}\mathbf{x_{k-1}} + \mathbf{B}\mathbf{U} \]

  1. Predict State Covariance.

\[\mathbf{P} = \mathbf{A}\mathbf{P}\mathbf{A}^{T} + \mathbf{Q} \]

  1. Update Kalman Gain

\[\mathbf{K} = \mathbf{P}\mathbf{C}^{T} (\mathbf{C}\mathbf{P}\mathbf{C}^{T} + \mathbf{R})^{-1} \]

  1. Correction Step (Posteriori State Estimate)

\[\mathbf{x} = \mathbf{x} + \mathbf{K} (\mathbf{y} - \mathbf{C}\mathbf{x}) \\ \mathbf{P} = (\mathbf{I} - \mathbf{K}\mathbf{C}) \mathbf{P} \]

Example

  1. Initialize model and filter

>>> import torch
>>> import pypose as pp
>>> model = pp.module.System()  # System model
>>> filter = pp.module.EKF(model)

  1. Calculate filter over one timestep

>>> # u is system input
>>> state, observation = model(state, u)  # model measurement
>>> model.set_refpoint(state=state, input=u)
>>> est_state, P = filter(est_state, observation, P, u)

Note

Implementation is based on section 5.1 of this book

  • Dan Simon, Optimal State Estimation: Kalman, H∞, and Nonlinear Approaches, Cleveland State University, 2006
