pypose.module.LTI

class pypose.module.LTI(A, B, C, D, c1=None, c2=None)[source]

A sub-class of ‘_System’ to represent Linear Time-Invariant system.

Parameters

  • A (Tensor) – The coefficient matrix in the state-space equation of LTI system

  • B (Tensor) – The coefficient matrix in the state-space equation of LTI system

  • C (Tensor) – The coefficient matrix in the state-space equation of LTI system

  • D (Tensor) – The coefficient matrix in the state-space equation of LTI system

  • c1 (Tensor) – Bias generated by system

  • c2 (Tensor) – Bias generated by system

Note

According to the actual physical meaning, the dimensions of A, B, C, D must be the same, whether in the batch case or not.

The system is time invariant.

forward(x, u)[source]
Parameters

  • x (Tensor) – The state of LTI system,

  • u (Tensor) – The input of LTI system.

Returns

The solved batched tensor, which means state-transition and observation of the system.

Return type

Tensor

Every linear time-invariant lumped system can be described by a set of equations of the form which is called the state-space equation.

\[\begin{align*} z_{i} = A_{i} \times x_{i} + B_{i} \times u_{i} + c_1 \\ y_{i} = C_{i} \times x_{i} + D_{i} \times u_{i} + c_2 \\ \end{align*} \]

where \(x\) is actually \(\dot{x}\), the differential form of \(x\).

Let the input be matrix \(A\), \(B\), \(C\), \(D\), \(x\), \(u\), \(c_1\), \(c_2\). \(x_i\) represents each individual matrix in the batch.

Note

-\(x\), \(u\) could be single input or multiple inputs

-\(A\), \(B\), \(C\), \(D\) can only be two-dimensional matrices or the batch

In the batch case, their dimensions must be the same as those of \(x\), \(u\). \(A\), \(B\), \(C\), \(D\) and \(x\), \(u\) are multiplied separately for each channel.

-For a System with p inputs, q outputs, and n state variables, \(A\), \(B\), \(C\), \(D\) are n*n n*p q*n and q*p constant matrices.

-Note that variables are entered as row vectors.

Example

    B = torch.randn((5,4,2))

    D = torch.randn((5,3,2))

    x = torch.randn((5,1,4))
