pypose.mul

class pypose.mul(input, other)[source]

Multiplies input LieTensor by other.

\[\bm{y}_i = \bm{x}_i \times \bm{a}_i \]

where \(\bm{x}\) is the input LieTensor, \(\bm{a}\) is the other value, and \(\bm{y}\) is the output value.

LieTensor @ LieTensor -> LieTensor (group multiplication)

LieTensor @ Tensor -> Tensor (group action)

Parameters

  • input (LieTensor) – the input LieTensor (Lie Group or Lie Algebra).

  • other (Tensor or LieTensor) – the value for input to be multiplied by.

Returns

the product with the same type as other

Examples

>>> x = pp.randn_SO3()
>>> a = torch.randn(4)
>>> x * a
tensor([ 0.2354, -0.9344, -0.0361,  1.3125])
>>> x @ a
tensor([ 0.2354, -0.9344, -0.0361,  1.3125])
>>> pp.mul(x, a)
tensor([ 0.2354, -0.9344, -0.0361,  1.3125])

>>> a = pp.randn_SO3()
>>> x * a
SO3Type LieTensor:
LieTensor([-0.7324, -0.4761, -0.2069,  0.4406])
>>> x @ a
SO3Type LieTensor:
LieTensor([-0.7324, -0.4761, -0.2069,  0.4406])
>>> pp.mul(x, a)
SO3Type LieTensor:
