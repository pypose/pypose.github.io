Key Features &
Capabilities

PyTorch Integration

PyPose is built upon the PyTorch library with deep integrations with the functionality and syntax you know and love.

Robotics-Oriented

Designed for robotic applications including SLAM (VIO, stereo VO, multiple view VO, LiDAR), Control (MPC), and Planning.

Geometric Learning

Can combine geometry-based methods such as bundle adjustments and factor graph optimization with learning-based methods such as feature extraction and loop closure detection.
